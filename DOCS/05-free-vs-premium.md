## Plan Upgrade Path

```
Free → Pro      : Instant (Stripe payment)
Pro → Enterprise: Instant (Stripe payment, prorated)
Pro → Free      : Downgrade at next billing cycle
Enterprise → Pro: Downgrade at next billing cycle
```

Downgrade limits are enforced at billing cycle end:
- Products over limit → oldest products moved to draft
- Imports over limit → imports remain but new imports blocked
- Streams over limit → future streams blocked until next period
